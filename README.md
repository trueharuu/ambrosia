# ambrosia
An experimental hexagonal tic-tac-toe bot written in Rust.

## Prerequisites
This bot requires Rust to be installed. You can find an installation method for `rustup` and `cargo` here: https://rust-lang.org/
## Running
This engine is currently very limited. You can simulate a game of `heuristic` vs. `heuristic` with the following command:
```
cargo r -r --bin harness
```

## Naming
Bees like hexagons, and bees like pollen. Ambrosia happens to be one of the plants that produces the most pollen.
